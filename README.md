# Paint-Program

This paint program includes the following:

    -Drawing brush, square, rectangle, circle, 2D polygon, eraser
  
    -Change canvas colour
  
    -Colour palette
  
    -Changeable line thickness
  
    -Fill option
  
    -Save/Load feature
  
    -New/clear canvas feature
